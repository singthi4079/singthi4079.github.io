字符数据类型和操作

字符数据类型(char)表示单个字符. 字符型直接量用单引号括住.

字符串直接量必须括在双引号(“”)中.

一个字符在计算机中是以0和1构成的序列形式存储的, 将字符映射到它的二进制形式的过程称编码.

Unicode码占两个字节, 以\u开头的4位十六进制数表示.

转义序列有反斜杠(\)后面加上一个字符或者一些数字组成.

char型数据可以转换成任意一种数值类型, 反之亦然.

String类型
