Hexo+GitHub搭建属于自己的博客

hexo是一款基于Node.js的静态博客框架.
需要配置的环境:

  1. Node
  2. Git
  3. GitHub账号
  4. Hexo

安装Hexo

使用一下命令安装Hexo
npm install -g hexo

hexo init初始化hexo

hexo generate或者hexo g生成静态页面

hexo server启动本地服务, 在浏览器中输入http://localhost:4000查看

配置GitHub

在GitHub创建一个新的Respository, 仓库名必须是

修改_config.xml文件, 建立与GitHub仓库的关联
在_config.xml文件的最后, 修改deploy下的内容
deploy:
type: git
repo: GitHub仓库地址
branch: master

安装hexo-deployer-git
npm install hexo-deployer-git --save

执行hexo deploy命令将项目部署到GitHub上
在浏览器中输入: https://GitHub仓库名

部署三部曲

  1. hexo clean
  2. hexo generate
  3. hexo deploy

Hexo 常用命令

  1. hexo new "postName"– 新建文章
  2. `hexo new page “pageName” – 新建页面
  3. hexo generate – 生成静态页面到public目录下
  4. hexo server – 开启本地预览
  5. hexo deploy – 将.deploy目录部署到GitHub
  6. hexo help – 查看帮助
