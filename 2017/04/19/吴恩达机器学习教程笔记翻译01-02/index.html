什么是机器学习(What is machine learning)

  • 在这里(Here we..)
    • 定义(Define what it is)
    • 在什么时候使用(When to use it)
  • 一个不是很明确的定义(Not a well defined definition)
    • 人们尝试如何定义它的几个例子
  • Arthur Samuel (1959)
    • 机器学习: “机器学习的领域, 无需明确的编程”(Machine learning: “Field of study that gives computers the ability to learn whithout being explicitly programmed”)
      • Samuels写了一个跳棋程序(Samuels wrote a checkers playing program)
        • 这个程序和自己玩了10000场游戏(Had the program play 10000 games against itself)
        • 更具胜/负, 觉得哪些位置的好坏(Work out which board positions were good and bad depending on wins/losses)
  • Tom Michel (1999)
    • 良好的学习问题:”A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.”
      • 以跳棋为例(The checkers example),
        • E表示10000次游戏(E = 10000’s games)
        • T表示玩跳棋(T is playing checkers)
        • P表示输赢(P if you win or not)
  • 几种学习算法(Several types of learning algorithms)
    • 监督学习(Supervised learning)
      • 教电脑如何做某事, 然后让它使用它的新知识来做到这一点(Teach the computer how to do something, then let it use it’s new found knowledge to do it)
    • 无监督学习(Unsupervised learning)
      • 让计算机学习如何做某事, 并用它来确定数据中的结构和模式(Let the computer learn how to do something, and use this to determine structure and patterns in data)
    • 加强学习(Reinforcement learning)
    • 推荐系统(Recommender systems)
  • 本课程(This course)
    • 学习算法的实用建议(Look at practical advice for applying learing algorithms)
    • 学习一套工具以及如何使用它们(Learning a set of tools and how to apply them)

监督学习介绍(Supervised learning - introduction)

  • 可能是机器学习中最常见的问题类型(Probably the most common problem type int machine learning)
  • 从一个例子开始(Starting with an example)
    • 我们如何预测房价(How do we predict housing prices)
      • 收集关于房价的数据, 以及它们与尺寸的关系(Collect data regarding housing prices and how they relate to size in feet)

  • 实例问题:鉴于这些数据, 一个朋友有一个房子750平方英尺-它们可以得到多少钱?(Given this data, a friends has a house 750 square feet - how much can they be expected to get)

  • 我们用什么方法来解决这个问题(What approaches can we use to solve this)

    • 直线通过数据(Straight line through data)
      • 可能 $150000(Maybe $150000)
    • 二阶多项式(Second order polynomial)
      • 可能 $200000(Maybe $200000)
    • 有件事我们稍后讨论 - 如何选择直线或曲线(One thing we discuss later - how to chose straight or curved line?)
    • 每一种方法都代表监督学习的一种方式(Each of these approaches represent a way of doing supervised learning)

  • 这是什么意思?(What does this mean?)

    • 我们通过算法提供了这些数据集的”正确答案”(We gave the algorithm a data set where a “right answer” was provided)
    • 所以我们知道房子的实际价格(So we know actual prices for houses)
      • 我们可以从训练数据中得到的想法(The idea is we can learn what makes the price a certain value from the training data)
      • 然后, 该算法应该根据新的训练数据产生更多的正确答案, 我们不知道这个价格.(The algorithm should then produce more right answers based on new training data where we don’t know the price already)
        • 即预测价格

  • 我们也称之为回归问题(We also call this a regression problem)

    • 预测连续产值(价格) (Predict continuous valued output(price))
    • 没有真正的离散描绘(No real discrete delineation)

  • 另一个例子(Another example)

    • 我们可以根据肿瘤大小定义乳腺癌是恶性还是良性(Can we difner breast cancer as malignant or benign based on tumour size)

  • 观察数据(Looking at data)

    • 每次五个
    • 你可以根据肿瘤大小估计后面的数值吗?(Can you estimate prognosis based on tumor size?)
    • 这是一个分类问题的例子(This is an example of a classification problem)
      • 将数据分为两个离散类之一 - 无论是否是恶性
      • 在分类问题中, 可以有多个离散的输出值(In classification problems, can have a discrete number of possible values for the output)
        • 例如: 可能有四个值
          • 0 - benign
          • 1 - type 1
          • 2 - type 2
          • 3 - type 3

  • 在分类问题中, 我们可以用不同的方式绘制数据

  • 只使用一个属性(大小) (Use only one attribute(size))
    • 其他问题可能有多个属性(In other problems may have multiple attributes)
    • 例如, 我们也可能知道年龄和肿瘤大小(We may also, for example, know age and tumor size)

  • 基于这些数据, 你可以尝试通过一下方式定义单独的类
    • 在两组之间画一条直线(Drawing a straight line between the to groups)
    • 使用更复杂的函数来定义两个组(稍后讨论) (Using a more complex function to define the tow groups (which we’ll discuss later))
    • 然后, 当你具有特定肿瘤大小, 特定年龄的个体时, 你希望能将这些信息放入到你的一个类别中.(Then, when you have an individual with a specific tumor size and who is a specific age, you can hopefully use that information to place them into one of your classes)
  • 你可能需要考虑许多特征(You might have many features to consider)
    • 团块厚度(Clump thickness)
    • 细胞大小均匀(Uniformity of cell size)
    • 细胞形状均匀(Uniformity of cell size)
  • 最令人兴奋的算法可以处理无限数量的特征(The most exciting algorithms can deal with an infinite number of features)
    • 你如何处理无限数量的特征?(How do you deal with an infinite number of features?)
    • 支持向量机的整齐数学技巧(稍后讨论) (Neat mathematical trick in support vector machine)
      • 如果你有一个无限长的列表 - 我们可以开发一个算法来处理(If you have an infinitely long list - we can develop an algorithm to deal with that)
  • 概要(Summary)
    • 监督学习可以让你获得”正确”的数据(Supervised learning lets you get the “right” data)
    • 回归问题(Regression problem)
    • 分类问题(Classification problem)

无监督学习介绍(Unsupervised learning - introduction)

  • 第二大问题类型(Second major problem type)
  • 在无监督学习中, 我们得到了未标记的数据(In unsupervised learning, we get unlabeled data)
    • 刚刚才说 - 这是一个数据集, 你可以构造它(Just told - here is a data list, can you structure it)
  • 这样做的一种方法是将数据聚类到组中(One way of doing this would be to cluster data into to groups)
    • 这是一个聚类算法(This is a clustering algorithm)

聚类算法(Clustering algorithm)

  • 聚类算法的例子(Example of clustering algorithm)
    • Google news
      • 将新闻报导成凝聚力集团(Groups news stories into cohesive groups)
    • 用于任何其他问题(Used in any other problems as well)
      • 基因组学(Genomics)
      • 微阵列数据(Microarray data)
        • 有一群人(Hava a group of individauls)
        • 基因的每个测量表达(On each measure expression of a gene)
        • 运行算法将个体聚类成人类(Run algrithm to cluster individuals into types of people)
        • 组织计算机集群
          • 识别潜在的弱点或有效分配工作
        • 社交网络分析
          • 客户数据
        • 天文数据分析
          • 算法给出惊人的结果
  • 基本上
    • 你能自动生成结构
    • 因为我们不给它答案, 这是无监督学习
