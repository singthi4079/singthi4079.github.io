Java重入锁源码分析

Posted on 2017-10-12

ReentrantLock类的声明如下:

实现了Lock接口.

ReentrantLock类内有定义了三个内部类: Sync, NonFairSync, FairSync;

Sync类提供所有的同步机制的实现.

NonFairSync是Sync类的子类, 表示非公平锁.

FairSync也是Sync类的子类, 表示公平锁.
